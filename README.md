📊 Dashboard CA - Furnishing & Houseware
Dashboard de gestão de Carteira Assessorada (CA) para a vertical F&H do MercadoLibre Brasil.
🎯 O que é este dashboard?
Painel unificado para acompanhamento diário de:

NMV (Net Merchandise Value)
NSI (Número de Itens Vendidos)
NASP (Net Average Selling Price)
Rebates e Investimentos Comerciais
📋 Estrutura do Painel
O dashboard possui 5 abas principais:

Aba
Descrição
Summary CA
Visão geral CA Total / Gestão F&H / Others
Gestão F&H
Performance da carteira MeliPro
Others
Sellers CA fora da carteira F&H
Diarizado
Performance diária com filtros
Setup
Configurações (Equipe, Sellers, Metas)
